# ProductFinder

Python GUI APP that gathers data on newegg products and outputs an excel sheet with that data.
The following data is gathered:
<ul style="list-style: none;">
  <li>hyperlink</li>
  <li>details</li>
  <li>price</li>
  <li>shipping price</li>
  <li>number of reviews</li>
  <li>page found</li>
 </ul>
